# chess_game

**Chess game made using flutter**

## Requirements

- Flutter SDK (v3.0+)
- Android Studio or VS Code
- Emulator or physical device

## Folder Structure

├── lib/
│ ├── custom/
│ │ ├── piece.dart
│ │ ├── square.dart
│ ├── easyconst/
│ │ ├── color.dart
│ ├── helper/
│ │ ├── helper_methods.dart
│ ├── screens/
│ │ ├── gamepage.dart
│ │ ├── splash.dart
│ ├── main.dart

## How to Run

1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`
